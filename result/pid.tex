\subsubsection{Iteration 1: PID-based Control for Path Following}

This iteration investigates the feasibility of using a classical PID controller as a Human-AI Collaboration (HAC) agent for assisting drone path following.

\paragraph{Design}

As this iteration focuses on control algorithm behavior rather than immersive interaction, the experiment was conducted in a flat panel environment. Instead of using the DroneXrEmulator (see \cref{fig:quad_structure}), a simplified PidDroneEmulator was implemented.

The PidDroneEmulator maps the PID controller outputs—represented as normalized joystick rates—directly to the YueInputModule. This abstraction allows the controller to operate under the same input interface as a human pilot, ensuring consistency between AI-generated and human-generated control signals.

The PID controller was designed to minimize the positional error between the drone and a predefined waypoint path, producing continuous joystick corrections for roll, pitch, and throttle.

\paragraph{Enactment}

During execution, the PID controller continuously computed normalized joystick inputs based on the real-time positional error.

These computed joystick rates were visualized on a HUD (see \cref{fig:pid-ui}), allowing human users to observe the controller's intended actions. This visualization acts as an AI collaboration hinter, exposing the controller's decision-making process and providing guidance on optimal stick positions for maintaining the designated path.

\paragraph{Analysis}

The PID controller demonstrated the ability to autonomously guide the drone from its initial position to the designated waypoint and return along the path. This indicates that the controller is functionally capable of performing the path-following task.

However, a critical limitation was observed in the controller's interaction characteristics. The PID controller produced rapid and frequent changes in joystick outputs, resulting in high-frequency control signals that are difficult for human users to interpret and replicate in real time.
