\subsection{Limitations}

% physics
\subsubsection{Self-Leveling Flight Mode Only}

For implementation simplicity, the current system is limited to self-leveling flight dynamics. Although Yue's Ultimate Drone physics model supports both self-leveling and acrobatic modes, this study focuses exclusively on the former to align with the design of the human-AI collaboration agents, which are optimized for stable flight control.

Consequently, acrobatic behaviors are not supported in the current implementation, which restricts the system's applicability to advanced flight scenarios.

% xr
\subsubsection{Visual Feedback Only}

Extended reality (XR) systems can support multimodal feedback. Additional modalities, such as haptic feedback and spatial audio, were considered in the design phase of the human-AI collaboration system, but they were not implemented in the current system.

This decision was driven by the additional design complexity associated with determining appropriate feedback types, intensity levels, and integration strategies. These aspects remain open research challenges for future work.

\subsubsection{No Participant Testing}

This study primarily focuses on system design and implementation rather than controlled user evaluation. As XR-based control interfaces are still relatively unfamiliar to many users, variability in prior experience may significantly influence task performance.

This makes it difficult to isolate learning effects from interface unfamiliarity, thereby limiting the interpretability of the user study results.
